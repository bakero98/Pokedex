"# Pokedex" 

Example app that uses : 

- MVVM
- Dagger 2
- ViewModel
- View Binding
- ROOM
- Type Converters
- ViewPager 2
- Recycle View
- Shared Element from Recycle View
- Bottom Navigation View with Floating Action Button
